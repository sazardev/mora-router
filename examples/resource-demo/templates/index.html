Mora Router Resource Demo

This is a demonstration of the Resource functionality in Mora Router.

Available Endpoints

Method URL Action Description
GET {{.BasePath}} Index List all users
GET {{.BasePath}}/:id Show Get a single user by ID
POST {{.BasePath}} Create Create a new user
PUT {{.BasePath}}/:id Update Update an existing user
DELETE {{.BasePath}}/:id Delete Delete a user

Example Usage

List all users

GET {{.BasePath}}

Get a specific user

GET {{.BasePath}}/1

Create a new user

POST {{.BasePath}}
Content-Type: application/json

{
  "name": "John Doe",
  "email": "john@example.com"
}

Update a user

PUT {{.BasePath}}/1
Content-Type: application/json

{
  "name": "Jane Doe",
  "email": "jane@example.com"
}

Delete a user

DELETE {{.BasePath}}/1
